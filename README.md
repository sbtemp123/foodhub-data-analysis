# foodhub-data-analysis
FoodHub Demand Analysis – Data Science
## Project Overview
This project analyzes FoodHub’s food delivery data to understand customer ordering behavior, restaurant popularity, and delivery efficiency in New York. It examines order cost, food preparation time, delivery time, and customer ratings across weekdays and weekends. The insights help FoodHub improve delivery speed, restaurant performance, and overall customer experience.
## Key Features
•	Analyzed food delivery data to uncover customer behavior and demand trends.
•	Performed EDA and created visualizations to identify peak demand and top-performing restaurants.
•	Provided data-driven recommendations to improve operational efficiency and user experience.

## Tech Stack
Python, Pandas, Data Visualization, Data Cleaning & Descriptive Statistics, Data Visualization, EDA
